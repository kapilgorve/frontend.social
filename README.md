# Frontend-social

Frontend Social website

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Local Dev
Use github to login for local development on the frontend app.

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```

### Run your unit tests

```
npm run test
```
